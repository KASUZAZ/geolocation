Projek ini adalah alat penjejakan lokasi ringkas menggunakan Python yang menukar koordinat GPS (Latitude & Longitude) kepada alamat jalan yang spesifik. Dibangunkan sebagai sebahagian daripada portfolio teknikal saya dalam bidang Pentadbiran Sistem Komputer.

## 🚀 Ciri-ciri Utama
* **Ketepatan Tinggi:** Mendapatkan data lokasi yang spesifik sehingga ke nama jalan dan poskod.
* **Integrasi Geopy:** Menggunakan library `geopy` untuk proses *reverse geocoding*.
* **Output Jelas:** Paparan lokasi dalam format yang mudah dibaca.

## 🛠️ Prasyarat & Pemasangan

Pastikan anda mempunyai Python dipasang pada sistem anda (Sangat disyorkan menggunakan Kali Linux).

1. **Clone repositori ini:**
   ```bash
   git clone [https://github.com/KASUZAZ/geolocation.git](https://github.com/KASUZAZ/geolocation.git)
   cd geolocation

2. pip install geopy
3. python3 location.py
